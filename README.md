# Taiwan Government Open Datasets

# 計畫目標
- 彙整並建立台灣政府相關基礎資訊開放資料集
- 提供一共同的參考資料集

# 計畫範圍

- 中央政府
- 地方 22 縣市
- 其他法人與國營企業
